# Study Capsules

This directory organizes the repository by claim and evidence bundle rather than by file type.

- [foundation-pole-shadow/README.md](./foundation-pole-shadow/README.md): the original pole-shadow walkthrough, baseline scripts, and baseline evidence exports.
- [settling-time-blind-spot/README.md](./settling-time-blind-spot/README.md): the matched-settling hidden-cost concept and its latent-detector study.
- [shadow-mass-saturation-threshold/README.md](./shadow-mass-saturation-threshold/README.md): the concept note on noise-conditioned shadow-mass sweet spots.
- [feedback-measurement-noise-phase-transition/README.md](./feedback-measurement-noise-phase-transition/README.md): the follow-up sensor-noise phase-transition concept and its supporting study.
- [out-of-family-plant-pi-validation/README.md](./out-of-family-plant-pi-validation/README.md): the first explicit plant-plus-PI out-of-family validation of the framework.

Each study keeps its concept note, scripts, and generated outputs together under one folder.
